## Installation

```sh
npm install
```

## Create env file (example below for local)

```sh
copy .env.template .env
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
